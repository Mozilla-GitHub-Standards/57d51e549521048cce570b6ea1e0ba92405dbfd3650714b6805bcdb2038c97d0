These are the developer docs for the Firefox Accounts email-sending service. For higher-level documentation, see the readme.

The project is compiled as a library that is linked against by two separate binaries:

  • fxa_email_send runs a Rocket server exposing an endpoint that enables callers to send email.

  • fxa_email_queues runs a process that loops infinitely, polling SQS queues for SES bounce, complaint and delivery notifications.
